This is Sc21-1.0beta by Systems in Motion. Double-click on Sc21.mpkg
to start installation. The installer includes Coin-2.3.0.

Note that this Coin framework has been built with gcc 3.3 and will
only work on Mac OS 10.3 "Panther" and above.

The final release will also be available for Mac OS 10.2
"Jaguar"

Check out http://www.coin3d.org/mac for information on how to get
started using Coin and Sc21.

Enjoy!
