This is Sc21 V1.0 by Systems in Motion. Double-click on Sc21.mpkg
to start installation. The installer includes Coin-2.3.0.

Note that these frameworks have been built with gcc 3.3, meaning they
should not be used in combination with any earlier version of gcc.
The frameworks is, however, built with MACOS_DEPLOYMENT_TARGET=10.2 to
be deployable on 10.2 (Jaguar) or later versions of Mac OS X.

Please make sure to also read the relevant FAQ entry at
http://www.coin3d.org/mac/faq.html#gcc_abi

Check out the following resources for information on how to get
started using Coin and Sc21:
o Coin/Mac developer page: http://www.coin3d.org/mac
o Sc21 API doc: http://doc.coin3d.org/Sc21
o Sc21 examples: /Developer/Examples/Sc21/

Enjoy!
